# 📝 Page and Pen

 A personal blog website where I write about things I'm learning, thinking about, and building , one page at a time.

---

## 🙋 About This Project

**Page and Pen** is my first personal blog website, built as a student to practice web development and share my journey. This project helped me learn how to design and build a real website from scratch.

---

## ✨ Features

- 📄 Clean, readable blog post layout
- 🏠 Home page with list of recent posts
- 📱 Responsive design (works on mobile & desktop)
- 🎨 Minimal and distraction-free design

---

## 🛠️ Built With

- Embedded Javascript
- JavaScript
- Node.js,Express.js 
- PostgreSQL

---

## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/Janani-bn/Page-and-Pen.git

# Navigate into the folder
cd Page-and-Pen

# Open in browser
nodemon index.js
```
